Java基础知识之基本程序设计结构

字符串

Java中包含三种处理字符串的类，分别是String，StringBuilder，StringBuffer。

String

String的底层是final char[]的数组，所以无法修改其中的字符，但可以做连接、切片等一些操作。

一些常用的API接口

  • 返回给定位置的字符。
  • 返回给定位置的字符的Unicode值。
  • 返回从startIndex码点开始，cpCount个码点后的码点索引。
  • 按照字典序，如果字符串位于other之前，返回一个负数；如果相同，返回0；如果位于other之后，返回一个正数。
  • 将这个字符串的码点作为一个流返回。调用toArray将他们放在一个数组中。
  • 用数组中从offset开始的count个码点构造一个字符串。
  • 如果字符串为空或由空格组成，返回true。
  • 如果字符串与other相同，返回true。
  • 如果字符串以prefix开头或以suffix结尾，返回true。
  • 返回与字符串str或码点cp匹配的第一个子串的开始位置，从索引0或fromIndex开始匹配。如果在原始字符串中没有str，返回-1。
  • 返回与字符串str或码点cp匹配的最后一个子串的位置，从原始字符串末尾或fromIndex开始匹配。
  • 返回字符串中代码单元的个数。
  • 返回startIndex和endIndex - 1之间的码点个数。
  • 返回一个新字符串。这个字符串用newString代替原始字符串中所有的oldString。可以用String或StringBuilder对象作为CharSequence参数。
  • 返回一个新字符串。这个字符串包含原始字符串中从beginIndex到字符串末尾或endIndex - 1的所有代码单元。
  • 返回一个新字符串。这个字符串将原始字符串的所有字母改成小写或大写。
  • 返回一个新字符串。这个字符串将删除原始字符串头部和尾部小于等于U+0020的字符trim，或空格strip。
  • 返回一个新字符串。用给定的定界符连接所有元素。
  • 返回一个字符串。将当前字符串重复count次。

StringBuilder

StringBuffer是StringBuilder的前身，由于StringBuffer允许多线程的方式添加或删除字符，所以效率较低，如果字符串的编辑操作都在单个线程中执行，则应该使用StringBuilder。两个类的API是一样的。

StringBuilder中的字符是可变的。有时需要创建的字符串较短，如果采用String拼接方式效率会比较低，因为每次拼接时都会构建一个新的String对象。使用StringBuilder类就可以避免此问题。在构建完成后调用toString就能得到String对象。

一些常用的API接口

  • 构造一个空的字符串构建器。
  • 返回构建器或缓冲器中的代码单元数量。
  • 追加一个字符串并返回this。
  • 追加一个代码单元并返回this。
  • 追加一个码点，并将其转换为一个或两个代码单元并返回this。
  • 将第i个代码单元设置为c。
  • 在offset位置插入一个字符串并返回this。
  • 在offset位置插入一个代码单元并返回this。
  • 删除偏移量从startIndex到endIndex - 1的代码单元并返回this。
  • 返回一个与构建器或缓冲器内容相同的字符串。

数组

数组拷贝

Java中允许将一个数组变量拷贝到另一个数组变量，这是两个变量将引用同一个数组。

此时如果修改了b中某个数，a数组也会一同被修改。

如果希望将一个数组中的所有值拷贝到新的数组中去，就要使用Arrays类的copy方法，第二个参数是新数组的长度。

数组排序

要想对数值型数组进行排序，可以使用Arrays类中的sort方法。

Arrays常用API

  • 返回包含a中所有元素的一个字符串。
  • 返回与a类型相同的一个数组，其长度为length或end - start，数组元素为a的值。如果end > a.length，结果会填充0或false。
  • 对数组进行排序，默认从小到大。
  • 使用二分查找在数组中查找元素v，如果找到，返回其下标；否则返回一个负数值r，-r-1是v应插入的位置。使用前要保证a有序。
  • 将数组所有元素值设为v。
  • 如果两个该数组大小相同且下标相同的元素相同，返回true。
