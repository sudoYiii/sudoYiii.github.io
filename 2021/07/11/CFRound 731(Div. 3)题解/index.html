Codeforces Round #731 (Div. 3)题解

传送门：Codeforces Round #731 (Div. 3) - Codeforces

A. Shortest Path with Obstacle

题意：给了三个点的坐标A,B,FA, B, FA,B,F，求AAA到BBB不经过FFF的最短距离。

思路：很容易发现，只有当A,BA, BA,B在同一行或同一列，且FFF在两点之间才会有影响。

B. Alphabetical Strings

题意：字符串sss会按照一定的规则形成：第iii步选择第iii个小写字母加入sss左边或右边。给了字符串sss，判断是否可以用此规则形成。

思路：可以逆着推，即考虑从第nnn个字母开始删除，利用双指针，如果最后l>rl>rl>r则合法，否则不合法。

C. Pair Programming

题意：有一份已经有kkk行的文档，两人会在文档上进行操作，并给出了两人的操作序列，他们可以轮流操作，如果为000说明新增一行，否则如果为xxx，说明修改第xxx行，判断两人的操作序列是否合法，如果合法，输出他们两人合起来的操作序列，否则输出−1-1−1。

思路：利用双指针，如果某人当前操作的值小于等于kkk行，那么就不断加入ansansans，如果他们的序列不合法，一定会是两人都无法操作的情况。

D. Co-growing Sequence

题意：称一个序列为成长序列，如果对于所有ai(1≤i≤n−1)a_i (1 \le i \le n - 1)ai​(1≤i≤n−1)，有ai & ai+1=aia_i \: \& \: a_{i+1} = a_iai​&ai+1​=ai​。现在给了你序列xxx，你需要找到字典序最小的序列yyy，使得xi⊕yi(1≤i≤n)x_i \oplus y_i (1 \le i \le n)xi​⊕yi​(1≤i≤n)是成长序列。

思路：有一个贪心的策略是尽量只改变后面的。分类讨论情况，当二进制下xix_ixi​的第jjj位为000，那么不管xi+1x_{i+1}xi+1​的第jjj位是什么，&\&&之后都为000，此时yi+1y_{i+1}yi+1​的第jjj位为000；如果xix_ixi​的第jjj位为111，且xi+1x_{i+1}xi+1​的第jjj位为000，那么就要让yi+1y_{i+1}yi+1​这一位为111， 否则为000。最后还要让xi+1x_{i+1}xi+1​去异或yi+1y_{i+1}yi+1​，否则后面计算会出问题。

E. Air Conditioners

题意：有kkk个空调，给了你位置和温度，每个位置定义了一个温度，计算公式为min1≤j≤k(tj+∣aj−i∣)min_{1 \le j \le k}(t_j + \mid a_j - i \mid)min1≤j≤k​(tj​+∣aj​−i∣)。求每个位置的温度。

思路：该公式即所有空调与某一位置的距离+该空调的温度的最小值。可以分析出，对于任意位置来说，它的温度是来自左边或者右边的空调，那么就可以利用两次dp​dp​dp​，一次从左向右，一次从右向左，最后两个方向里取最小的。例如从左向右的dp​dp​dp​，我们先初始化dp​dp​dp​数组为正无穷，如果该位置本身就有空调，我们可以将它先设置为该空调的温度，dp​dp​dp​方程就为dp[i]=min(dp[i−1]+1,dp[i])​dp[i] = min(dp[i - 1] + 1, dp[i])​dp[i]=min(dp[i−1]+1,dp[i])​，反之相同。

F. Array Stabilization (GCD version)

题意：给了一个数组aaa，可以执行以下操作任意次：ai=gcd⁡(ai,a(i+1)mod  n)(0≤i≤n−1)a_i = \gcd(a_i, a_{(i + 1) \mod n})(0 \le i \le n - 1)ai​=gcd(ai​,a(i+1)modn​)(0≤i≤n−1)，最少多少次可以使数组所有数相同。

思路：二分+线段树。先来看下对于x1x_1x1​经过几次gcdgcdgcd会变成什么，第一次：gcd(x1,x2)gcd(x_1,x_2)gcd(x1​,x2​)，第二次：gcd(x1,gcd(x2,x3))=gcd(x1,x2,x3)gcd(x_1, gcd(x_2, x_3)) = gcd(x_1, x_2, x_3)gcd(x1​,gcd(x2​,x3​))=gcd(x1​,x2​,x3​)，通过归纳法可以发现经过ttt步后x1x_1x1​会变成gcd(x1,x2,...,xt+1)gcd(x_1,x_2, ..., x_{t+1})gcd(x1​,x2​,...,xt+1​)，此时可以通过枚举要多少步可以使所有所有xix_ixi​相同，求区间gcdgcdgcd可以利用线段树，由于最多可能需要nnn步，所以还需要利用二分。后面几个数求区间gcdgcdgcd可能会不方便，即要查两次gcdgcdgcd，我们可以先复制一遍xxx数组，这样就只需查询一遍。

G. How Many Paths?

题意：给了一张有向图，判断111号点和每个点的关系：000——没有路，111——只有111条路，222——有多条且有限的路，−1-1−1——无穷的路。

思路：对于000很好判断，对于111和222，可以先用dfsdfsdfs去将所有能到达的点设置为111，如果途中经过了为111的点，则将那个点设置为222，然后再将所有222的点dfsdfsdfs一遍，该点为222，则它可以到达的所有点肯定都至少为222，对于−1-1−1，我们可以利用强连通分量缩点，在环上的点以及这个环能到达的所有点一定都是为−1-1−1。官方题解用了更加巧妙地dfsdfsdfs，可以帮助加强对于dfsdfsdfs的理解。
